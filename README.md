# ToolNest 🪺

> Everything a dev needs, nothing they don't.

ToolNest is a fast, no-fluff developer utility belt that lives in your browser. Stop Googling "base64 decode online" or hunting for a JSON formatter — just open ToolNest and get it done.

## What's inside

**JSON Formatter** — paste messy JSON, get it back clean. Supports formatting, minifying, validating, and custom indent levels.

**Base64** — encode or decode in one click. Handles UTF-8 properly. Swap output back to input when you need to chain operations.

**UUID Generator** — generate up to 50 v4 UUIDs at once. Toggle uppercase, strip dashes, copy one or all.

**Color Converter** — pick any color and instantly get HEX, RGB, HSL, RGBA, CSS variable, and Tailwind arbitrary class formats. Includes a 10-color quick-pick palette.

**Lorem Ipsum** — generate paragraphs, sentences, or words of placeholder text. Word and character count included.

## Getting started

```bash
git clone https://github.com/Muhammadwaseem1/toolnest.git
cd toolnest
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and you're in.

## Built with

- [Next.js 15](https://nextjs.org/) — App Router, no unnecessary complexity
- [TypeScript](https://www.typescriptlang.org/) — full type safety
- [Tailwind CSS](https://tailwindcss.com/) — utility-first styling
- Zero external runtime dependencies for the tools themselves — everything runs client-side

## Why ToolNest?

Every dev ends up with 15 browser tabs open to random "online tool" sites covered in ads. ToolNest is the opposite of that — dark, fast, clean, and self-hostable.
